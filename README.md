2. Írja a nevét div elemek közzé.

4. A doboz szegélye legyen 3 pixeles.

5. A doboz szegélye legyen #800080 színű.

6. A doboz belső margója legyen 17 pixel.

7. A doboz külső margója legyen 10%.
